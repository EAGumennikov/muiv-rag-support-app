Прототип веб-интерфейса

Система поддержки пользователей с RAG-подходом

Этот интерфейс является первым каркасом веб-сервиса над существующим RAG-ядром. Здесь будет выполняться поиск по корпусу знаний и формирование итогового ответа для оператора или пользователя.

Назначение системы

Прототип предназначен для демонстрации того, как retrieval и внешняя LLM могут использоваться для поддержки поиска решений по обращениям пользователей.
