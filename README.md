# MUIV RAG Support App

Прототип системы управления знаниями службы поддержки пользователей с применением RAG-подхода.

## Что реализовано
- нормализация HTML-корпуса в Markdown
- чанкинг корпуса
- построение эмбеддингов
- построение FAISS-индекса
- извлечение retrieval-слоя по пользовательскому запросу
- построение ответа
- интеграция с Yandex AI Studio

## Структура
- scripts/ - основные скрипты проекта
- config/ - шаблоны конфигурации
- docs/ - технические заметки по проекту

## Зависимости
См. requirements.txt

## Контуры работы

### Репозиторий
Репозиторий содержит только код проекта, конфигурационные шаблоны и техническую документацию.

### Локальный рабочий каталог
Runtime-данные, виртуальное окружение, секреты и крупные артефакты хранятся вне репозитория в отдельном рабочем каталоге.

### Synology
Исходные данные, заметки, черновики и эксплуатационные артефакты проекта хранятся на Synology. В целевом сценарии запуск решения предполагается выполнять в данном контуре.

## Важно
Тяжелые данные, секреты, индексы и runtime-артефакты в репозиторий не включаются.
